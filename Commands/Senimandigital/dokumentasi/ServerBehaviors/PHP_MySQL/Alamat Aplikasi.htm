Untuk mendapatkan daftar file pada sebuah Directory

Pertama-tama definisikan nama directory kemudian tentukan apakah kita akan mengambil pula data pada sub folder, tetapkan default directory lalu definisikan variabel apabila kita ingin mendapatkan directory dari sebuah variabel terakhir buat daftar extension file yang ingin diambil apabila hanya satu kita cukup mengisi dengan extensi misal [php] namun jika banyak extensi gunakan [php','txt] atau [php','txt','exe].
