PEMERIKSAAN QUERY DROPDOWN

  1. Dapatkan recordset yang digunakan pada dropdown.
  2. Dapatkan query recordset berdasarkan nama recordset.
  3. Lakukan perbadingan query recordset pada recordset dengan query yang ideal.
  4. Tampilkan peringatan:
    1. Query calon karyawan menampilkan seluruh data anggota, sedang data anggota aktif ialah data dengan type aktif.
